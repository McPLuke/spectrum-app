# Spectrum

Spectrum is a web application designed to help individuals describe their experiences, strengths, and support needs related to the autistic spectrum.

The platform focuses on self-representation rather than diagnosis.

Users can create profiles, answer trait questionnaires, and generate a shareable page describing how they experience the world.

---

## Goals

- Provide a strengths-focused representation tool
- Support accessibility-first design
- Allow individuals to share context with employers, educators, or collaborators

---

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

Future plans include:

- Supabase database integration
- Authentication
- Shareable public profile pages
- Accessibility preference system
